## Python Variables
Variables are used to store the value/ data in to the object.

There are total 4 types of variable in python.
1) String  Ex: "Hello"
2) Integer Ex: 0,1,66,999
3) Float Ex: 55.5, 4.9
4) Boolean Ex: True or False
